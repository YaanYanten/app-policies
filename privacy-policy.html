Política de Privacidad

Última actualización: [FECHA]

1. Información que Recopilamos

Nuestra aplicación recopila la siguiente información:

  • Información de autenticación: Tokens de acceso de YouTube y TikTok para publicar contenido en tu nombre
  • Información de video: Metadatos de los videos que eliges subir (título, descripción, etc.)
  • Información de usuario: ID de usuario y información básica del perfil de YouTube y TikTok

2. Cómo Usamos tu Información

Utilizamos tu información únicamente para:

  • Subir videos a tus cuentas de YouTube y TikTok según tu solicitud
  • Mantener la conexión con las APIs de YouTube y TikTok
  • Proporcionar funcionalidad básica de la aplicación

3. Almacenamiento de Datos

Los tokens de acceso se almacenan localmente en tu dispositivo usando cifrado DPAPI de Windows. No enviamos esta información a servidores externos.

4. Compartir Información

No compartimos, vendemos ni alquilamos tu información personal a terceros. La información solo se transmite directamente a YouTube y TikTok según tus acciones.

5. Seguridad

Implementamos medidas de seguridad apropiadas para proteger tu información:

  • Cifrado local de tokens usando DPAPI
  • Comunicación segura (HTTPS) con las APIs
  • No almacenamiento de credenciales en texto plano

6. Retención de Datos

Los tokens se mantienen hasta que:

  • Desautorices la aplicación desde YouTube/TikTok
  • Elimines la aplicación
  • Elijas desconectar las cuentas desde la aplicación

7. Tus Derechos

Puedes:

  • Revocar el acceso en cualquier momento desde la configuración de YouTube/TikTok
  • Solicitar la eliminación de datos locales
  • Ver qué información tiene la aplicación

8. APIs de Terceros

Esta aplicación utiliza:

  • YouTube Data API: Sujeta a los Términos de Servicio de YouTube API
  • TikTok API: Sujeta a los Términos de Servicio de TikTok for Developers

9. Cambios a esta Política

Nos reservamos el derecho de actualizar esta política. Los cambios significativos serán notificados a través de la aplicación.
