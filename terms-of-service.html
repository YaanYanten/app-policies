1. Aceptación de Términos

Al usar TikTok YouTube Connector ("la Aplicación"), aceptas estos términos de servicio. Si no estás de acuerdo, no uses la aplicación.

2. Descripción del Servicio

La Aplicación permite:

  • Conectar cuentas de YouTube y TikTok
  • Subir videos a ambas plataformas desde una interfaz unificada
  • Gestionar metadatos de videos (títulos, descripciones, etc.)

3. Requisitos de Uso

Para usar la Aplicación debes:

  • Tener al menos 18 años o el consentimiento de un padre/tutor
  • Poseer cuentas válidas en YouTube y TikTok
  • Cumplir con los términos de servicio de YouTube y TikTok
  • No usar la aplicación para contenido ilegal o inapropiado

4. Responsabilidades del Usuario

Eres responsable de:

  • Contenido: Todo el contenido que subas es tu responsabilidad
  • Derechos de autor: Asegurar que tienes derecho a publicar el contenido
  • Cumplimiento: Seguir las políticas de YouTube y TikTok
  • Seguridad: Mantener seguras tus credenciales de acceso

5. Limitaciones de la Aplicación

La Aplicación:

  • Está en desarrollo activo y puede tener errores
  • No garantiza la disponibilidad 24/7
  • Depende de las APIs de YouTube y TikTok
  • Puede tener limitaciones de funcionalidad

El manejo de datos está detallado en nuestra Política de Privacidad. Aspectos clave:
